## The Brief
You are the Lead UI Designer for a new AAA game. Design the "Hero Screen" — the interactive webpage where players choose their faction, class, or character before the game begins.

**The Vibe:** This is an interactive poster. It needs to drip with atmosphere. When a user hovers over a character, the layout should react, the typography should shift, and the unselected options should fade into the background.

## What You Turn In
Everything lives in the GitHub repo and ships together:

1. **Live Hero Screen URL** — Single-page interactive experience. Atmospheric hover states, masterful typography, CSS Grid layout. Layout reacts when user hovers over a character/faction.
2. **Design Intent** — Creative spec written *before* AI engagement. Color palette, typographic hierarchy, hover-state rules, mood. This is the standard against which you evaluate all AI output.
3. **Mermaid Diagram** — System flow: what receives input, how the system processes it, what it outputs. In your GitHub repo.
4. **AI Direction Log (3–5 entries)** — What you asked AI to do, what it produced, what you changed/rejected/kept and why. README, annotated commits, or sketchbook.
5. **Records of Resistance (3 moments)** — Three documented moments where you rejected or significantly revised AI output. What did AI produce? Why did you reject it? What did you do instead?
6. **Five Questions Reflection** — Short paragraph in README or sketchbook. Completed before submission.
